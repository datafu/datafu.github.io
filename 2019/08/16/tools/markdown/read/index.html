    链接和图片

    在 Markdown 中插入链接不需要其他按钮，你只需要使用［显示文本］(链接地址)这样的格式语法即可。例如：
    链接

    插入图片的语法与插入链接的语法很像，只是前面多了一个 !.语法如下：
    ![图片](图片链接地址)

    引用
