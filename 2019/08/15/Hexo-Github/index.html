    mac下搭建Hexo-Github博客

    大前端/ web hexo
    创建时间:2019-08-15 10:10

      1. 安装 Hexo
      2. 创建博客
      3. 调试

    2. 参考该教程进行的博客搭建

    我们主要是使用 GitHubPages + Hexo的方式来创建免费的博客网站,做为世界上最大的源码托管平台,github越来越受到开发者的喜欢了,所以直接基于它的产品来搭建自己的博客平台也便于自己的代码托管和文档能够尽量保持一致,而不是东放一处,西放一处的。

    Hexo基本命令

    安装 Hexo

    • 安装 hexo $ npm install -g hexo

      创建博客

    • 使用命令 hexo init Blog 就创建blog项目了

    • 编写一个文章,使用 hexo new initBlog,就可以在Blog/source/中看到了 initBlog.md的文件了,然后就可以写md内容了

      调试

      我们使用 hexo server 就可以启动服务,http://localhost:4000/ 中就可以在浏览器中查看该博客内容页了

      发布

      • 创建仓库和安装插件
        发布到github中,为了方便传输,我们一般使用插件的形式进行源码的传输,
        $ npm install hexo-deployer-git --save,该插件自动部署到Github上。 但是我们知道，上传到github中,我们首先需要创建一个创库才行，
      • 如图:*

    • 安装发布插件 在博客文件夹运行下面命令 $ npm install hexo-deployer-git –save 然后在_config.yml文件修改发布的配置，最后一行改为（注意替换yourusername）

    • 接着我们在生产和发布

    • 快速发布

      hexo clean && hexo g && hexo d

    • 最后我们可以用过 git@github.com:yourusername/yourusername.github.io.git

      创建的仓库名要以用户名.github.io形式才行,不然到时候发布到github是不会有效果的.

    可以从以下方式进行主题的替换

    • https://hexo.io/plugins/ 绝对可以找到你所喜欢的主题
