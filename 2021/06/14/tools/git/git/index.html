    1. 配置账户

      git config. –global. –list

    • 添加文件

    Git add readme

    git commit -m ‘Add readme’

    Git status

    Git log

    //之前已经提交的全部提交到暂存区

    git add -u

    git status

    //给文件名重命名到方式

    git rm readme.md

    git add readme.md

    //暂存区清空

    Git reset

    //给文件重命名

    Git mv read readme.md

    //分支

    Git branch -av

    //切换分支

    git check master

    密钥对

    ssh-keygen -t rsa -b 4096 -C “your_email@example.com“

    git config –global user.name “fushuping”
    git config –global user.email “fushp@ffcs.cn“

    git config –global -unset-all

    清空配置

    git config –global –unset user.name
    git config –global –unset user.email

    git 也要设置代理才行
    git config –global http.proxy 192.168.13.19:7777

    取消代理:
    git config –unset http.proxy
    git config –global –unset http.proxy
